list - TumpCpp リファレンス

namespace tump {
    template <class... Types>
    struct list {};
}

概要

型のテンプレートパラメータパックTypesに指定した、任意の数の型を要素として保持します。

使い方

tump::listは複数の型をまとめて扱ったり、一括で処理を適用したい場合に使います。
要素が、値か型かの違いはありますが、std::vectorやstd::listと同じような用途です。

TumpCpp では、tump::listを主な対象とした、いくつかのリスト操作用メタ関数が定義されており、それらと組み合わせて使います。

また、型のみをテンプレートパラメータとして受け取る、任意のテンプレート型も、tump::listと同様に型のリストとして扱うことができます。

リスト用メタ関数・概念

tump::listおよび、型テンプレートパラメータを持つ、任意のテンプレート型に対して使用するためのメタ関数の一覧です。

情報取得

メタ関数・概念の名称 説明
tump::is_empty リストが空か判定します。
tump::is_type_list 型のリストとして利用できるテンプレート型かどうか判定します。
tump::TypeList 型のリストとして利用できるテンプレート型かどうか判定する制約です。
tump::TypeListOrValueList 型または、値のリストとして利用できるテンプレート型かどうか判定します。
tump::len リストの長さを取得します。

要素アクセス

メタ関数の名称 説明
tump::count_if 指定の条件に合致する要素の数を数えます。
tump::exists リストの要素に指定の型が含まれるかどうか判定します。
tump::find_if リストの要素から、指定の条件に合致する最初の型を取得します。
tump::get リスト要素のうち、指定の位置の型を取得します。
tump::get_front リスト要素の先頭の型を取得します。
tump::get_back リスト要素の最後尾の型を取得します。
tump::index_of リストの要素から、指定の型が最初に見つかった位置を取得します。
tump::index_of_if リストの要素から、指定の条件に合致する最初の型が見つかった位置を取得します。
tump::is_unique リストの要素の、重複の有無を判定します。

リスト操作

メタ関数の名称 説明
tump::make_empty 空のリストを作成します。
tump::make_type_list 新しいリストを作成します。
tump::concat 複数のリストを結合します。
tump::copy リストが持つテンプレートパラメータの内容を、別のリストに写します。
tump::filter リストが持つテンプレートパラメータから、条件に合致する要素のみ抽出します。
tump::foldl・tump::foldr 畳み込みを行います。
tump::map リストが持つ全ての要素に任意の処理を適用します。
tump::map_if リストが持つ要素のうち、指定の条件に合致する要素すべてに、任意の処理を適用します。
tump::pop_front リストの先頭の要素を削除します。
tump::pop_back リストの最後尾の要素を削除します。
tump::push_front リストの先頭に要素を挿入します。
tump::push_back リストの最後尾に要素を挿入します。
tump::push_front_if リストの先頭に、指定の条件に一致する要素を挿入します。
tump::push_back_if リストの先頭に、指定の条件に一致する要素を挿入します。
tump::remove_if リストが持つテンプレートパラメータから、条件に合致する要素のみ削除します。
tump::replace リストが持つ要素のうち、指定の型を新しい型で置き換えます。
tump::replace_if リストが持つ要素のうち、指定の条件に合致する型を新しい型で置き換えます。
tump::reverse リストの要素の並びを反転します。
tump::slice リストの指定の範囲を切り出します。
tump::unique リストの重複している要素を削除します。
tump::zip 2 つのリストから要素を 1 つずつ取り出した組のリストを生成します。

その他

メタ関数の名称 説明
tump::fmap リストがファンクタとして振舞うためのメタ関数です。機能はtump::mapとまったく同じです。
tump::pure メタ関数を、アプリカティブファンクタに適用可能な形に持ち上げます。
tump::ap リストがアプリカティブファンクタとして振舞うためのメタ関数です。アプリカティブファンクタに持ち上げられたメタ関数を、アプリカティブファンクタに適用します。
tump::ret メタ関数を、モナドに適用可能な形に持ち上げます。tump::pureの別名です。
tump::bind モナドから値を取り出し、モナドを返却するメタ関数に食わせます。

例

#include <tuple>
#include <type_traits>
#include <tump.cpp>

// map でテンプレートパラメータパックに一括適用。
// 型のテンプレートパラメータのみとるテンプレート型であれば、
// std::tuple や、std::is_same のようなテンプレート型でもOK。
using list1 = tump::eval<tump::map, tump::add_const, tump::list<int, float, char>>;
using list2 = tump::eval<tump::map, tump::add_const, std::tuple<int, float, char>>;
using list3 = tump::eval<tump::map, tump::add_const, std::is_same<int, float>>;

// 全てのテンプレートパラメタに const がついていることの確認。
static_assert(std::is_same_v<list1, tump::list<const int, const float, const char>>);
static_assert(std::is_same_v<list2, std::tuple<const int, const float, const char>>);
static_assert(std::is_same_v<list3, std::is_same<const int, const float>>);

int main() {}
